Running Multiple Models Concurrently

Ollama supports loading and running multiple models concurrently if your GPU has enough VRAM. To keep models loaded in memory, configure the environment variable OLLAMA_MAX_LOADED_MODELS=3 (or higher) in your system settings. Ollama automatically unloads models to free memory if new ones are requested.
